# 📄 GitHub Document Editor - Version Python/Qt

Une application Python avec interface graphique PyQt5 pour travailler sur des documents en ligne avec synchronisation GitHub.

## ✨ Fonctionnalités

- **Connexion sécurisée** à GitHub avec token d'accès personnel
- **Arborescence des fichiers** avec navigation intuitive
- **Éditeur de texte avancé** avec coloration syntaxique (Python)
- **Sauvegarde automatique** toutes les 30 secondes
- **Commit** direct vers GitHub avec message descriptif
- **Historique des commits** visible dans l'interface
- **Téléchargement** de fichiers vers le dépôt
- **Statistiques** en temps réel (lignes, mots, caractères)
- **Recherche et remplacement** de texte
- **Interface moderne** avec thème sombre/clair

## 📋 Prérequis

- Python 3.8 ou supérieur
- PyQt5
- Un compte GitHub
- Un token d'accès personnel avec permissions `repo` et `workflow`
